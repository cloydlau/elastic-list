# elastic-list / 动态列表&动态表格

![preview](./preview.gif)

### Feature

- √ v-model双绑
- √ 表格展示、列表展示两种展示方式
- √ 自定义列表 slot传入
- √ 对行进行增加、删除、拖拉拽排序
- √ 对行的数量上限、下限进行限制
- √ 自定义新增对象模板
- √ 全局安装/单独引入 通用参数支持全局配置

### Installation
![NPM](https://nodei.co/npm/elastic-list.png)
```
yarn add elastic-list
Dependencies: vue element-ui

全局引入：
import ElasticList from 'elastic-list'
Vue.use(ElasticList)

局部引入：
import { ElasticList } from 'elastic-list'
components: { ElasticList }
```
